## 中文

FinRead 是一个**单文件、纯浏览器运行**的英语精读工具。把你想读的英文文章粘贴进去，借助 AI 完成翻译、查词和精读，并把生词整理成可复习的词表，帮助你在真实阅读中积累词汇。

### 主要功能

- **粘贴即读**：直接粘贴任意英文文本，立即开始精读。
- **划词查词**：选中单词或短语即可查看释义、音标和例句。
- **AI 翻译**：整段或逐句翻译，对照理解。
- **朗读发音**：单词与句子语音朗读，辅助听力与跟读。
- **生词与掌握度**：标记生词、记录掌握程度，按需复习巩固。
- **离线运行**：核心界面为单个 HTML 文件，下载后可离线打开。
- **本地存储**：数据保存在你自己的浏览器中，隐私可控。
- **支持多种 AI 模型**：可配置 DeepSeek、Gemini 等 API。

### 使用方法

1. 打开 [在线地址](https://silangcloud-source.github.io/Finread/)，或下载 `FinRead 9 (offline).html` 双击在浏览器打开。
2. 在设置中填入你的 **AI API Key**（如 DeepSeek / Gemini）。
3. 粘贴英文文章，开始精读、查词、翻译与复习。

> 注意：AI 相关功能需要你自己的 API Key，密钥仅保存在本地浏览器，不会上传。

## English

FinRead is a **single-file, fully in-browser** English intensive-reading tool. Paste any English text, then use AI to translate, look up words, and read closely — while collecting unknown words into a reviewable list to build vocabulary from real reading.

### Features

- **Paste & read** — drop in any English text and start reading immediately.
- **Click-to-lookup** — select a word or phrase to see definitions, phonetics, and examples.
- **AI translation** — translate whole passages or sentence by sentence.
- **Text-to-speech** — read words and sentences aloud for listening practice.
- **Vocabulary & mastery** — flag new words, track mastery, and review as needed.
- **Works offline** — the core app is a single HTML file you can open without internet.
- **Local storage** — your data stays in your own browser.
- **Multiple AI models** — configure providers such as DeepSeek and Gemini.

### Getting started

1. Open the [live site](https://silangcloud-source.github.io/Finread/), or download `FinRead 9 (offline).html` and open it in your browser.
2. Enter your **AI API Key** (e.g. DeepSeek / Gemini) in the settings.
3. Paste an English article and start reading, looking up words, translating, and reviewing.

> Note: AI features require your own API key. The key is stored only in your local browser and is never uploaded.
